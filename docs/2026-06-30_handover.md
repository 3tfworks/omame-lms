   - `BookmarkGuide` 共有コンポーネント(`src/components/help/`)で help / LMS 両方から再利用

3. **proxy.ts:224 の `/login` 部分一致バグ修正(§7.5)**
   - `pathname.includes("/login")` だと `/ja/help/login` も誤マッチして強制リダイレクトされる事故を回避
   - 正規表現で厳密マッチに変更(`^\/[a-z]{2}\/login$`)
   - 6/29 障害復旧の独立コミットとして先行リリース

4. **git/本番 drift の意図せぬ解消**
   - §7.5 push 時に過去セッションで未 push だった5コミット(e8b508f / 308ab4a / e590e47 + docs2点)が一緒に乗った

   - 結果として **git remote main が本番実状態に追いついた**(初の正規 git-main 連動本番デプロイ)

5. **新ルール §5(f)「直 deploy 禁止」確立 + §5(c)「最終デプロイ ID 明示」初運用**
   - drift の根本原因が `vercel --prod` 直 deploy だったため、ルールとして禁止
   - 本日の本番反映が §5(c)(f) 両ルールの初運用例
