# 紹介購入10%OFF 設定手順

## 仕様

- 有効なサロンメンバー・管理者・オーナーの紹介リンク経由だけが対象
- 一般価格に10%割引を自動適用
- 購入者によるクーポンコード入力は不要
- 紹介購入時は他のプロモーションコードと併用不可
- 紹介報酬は割引後の実支払額を基準に計算
- サロン特別価格は紹介割引の対象外

## Stripeでの設定

1. Stripe Dashboardの「商品カタログ」から「クーポン」を開く
2. 割引率 `10%`、期間 `1回限り` のクーポンを作成する
3. 詳細画面に表示されるクーポンのAPI IDを控える
4. VercelのProduction環境変数へ次を追加する

```text
STRIPE_REFERRAL_COUPON_ID=クーポンのAPI ID
```

5. Productionを再デプロイする

クーポンIDが未設定の場合、紹介購入は通常価格に戻さず、安全のため購入画面の作成を停止する。
