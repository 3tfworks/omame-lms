# お豆メッセンジャー規約同意・成果管理 実装記録

作成日: 2026-07-16

## 決定した運用

- 2026年8月31日までは、紹介購入者に10%OFFを適用する。
- 2026年9月1日0時（日本時間）以降は、公式・紹介経由ともに29,800円とする。
- LINE、メール、対面などによる知人への個別紹介は、規約へ同意したサロンメンバーに許可する。
- ブログ、YouTube、公開SNSなど、不特定多数が閲覧できる場所での募集・リンク掲載は、運営が別途許可するまで禁止する。
- 紹介報酬50%は期間限定キャンペーンとする。キャンペーン終了後は管理画面で設定した通常報酬率を適用する。
- 紹介リンクの成果判定期間は30日とする。
- 購入後30日間は報酬の確認期間とする。
- 最低振込額は5,000円、月末締め・翌月末振込とする。
- 振込手数料は紹介者負担とし、報酬から差し引く。

## 実装内容

1. `affiliate_terms_acceptances` に、ユーザー、規約版、同意日時、重要確認文、規約全文のスナップショットを保存する。
2. 紹介ページでは4つの重要事項と規約全体への同意を必須にし、同意前は紹介リンクを表示しない。
3. 招待ページ、招待登録、紹介コード適用、Stripe決済、Stripe・会費ペイの報酬計上時に、現行規約への同意を確認する。
4. 2026年9月1日0時（日本時間）に、紹介10%OFFを自動停止する。
5. 報酬に支払可能日と取消状態を追加し、30日の確認期間前は支払済みにできないようにする。
6. 返金またはカード会社による支払いの取消をStripeから受信した場合、該当報酬を自動で取消状態にする。
7. 管理画面で規約同意者、同意日時、規約版、報酬の支払可能日・取消状態を確認できるようにする。
8. 振込CSVは、確認期間が終了し、紹介者ごとの合計が5,000円以上になった報酬のみを対象にする。

## 本番反映時の順序

1. Supabaseへ `20260716000000_affiliate_terms_acceptances.sql` を適用する。
2. Stripe Webhookの送信イベントに `charge.refunded` と `charge.dispute.created` を追加する。
3. アプリケーションをデプロイする。
4. 管理画面のキャンペーン設定で、50%キャンペーンの終了日時が `2026-09-01 00:00 JST` になっていることを確認する。
5. サロンメンバーへ新規約を案内し、既存のブログ・YouTube・公開SNS掲載があれば削除を依頼する。

アプリケーションを先にデプロイすると、同意テーブルが存在しないため紹介リンクが利用できない。必ずDBマイグレーションを先に適用する。
